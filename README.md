# Note app
Note app es una aplicacion web que permite crear y organizar notas de forma simple.

## Funcionalidades actuales
- Crear notas
- Consultar notas

## Tecnologias
- Frontend: HTML, CSS, JS
- Backend: FastAPI
- Base de datos: Pendiente

## Pasos pendientes
- Agregar la funcionalidad necesaria para editar y eliminar notas.
- Listar tareas como completadas o pendientes.
- Autenticacion y gestion de usuarios.
